### 构建

如果你需要重新编译项目，可以执行以下 2 步：

1. 克隆源码

```bash
$ git clone https://github.com/shiqinfeng1/goframe-ddd.git
```

1. 编译

首次运行时执行：

```bash
$ make tools
```

项目根目录下执行:

```bash
$ make
```

构建后的二进制文件保存在 `_output/platforms/<OS>/<ARCH>/` 目录下。
